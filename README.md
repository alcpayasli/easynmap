# easynmap
basic scans of nmap for kali linux
# Scans:
    1-INTENSE SCAN
    2-INTENSE SCAN (PLUS UDP)
    3-INTENSE SCAN (ALL TCP PORTS)
    4-PING SCAN
    5-QUICK SCAN
    6-QUICK SCAN (PLUS)
    7-VERSION AND OS SCAN
# Using:
   open terminal where it was loaded and run 'python easynmap.py' and start
